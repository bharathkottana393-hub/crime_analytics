# crime_analytics
 Aspiring Data Analyst | Power BI | Python | Data Visualization | Turning data into insights
📊 Crime Analysis Dashboard (Power BI)
🔍 Overview

This project analyzes crime data to identify patterns, trends, and hotspots using Power BI.

✨ Features
KPI Cards (Total Crimes, Arrests, Domestic Crimes)
Crime trend over time
Crime type analysis
Location-based map visualization
Interactive filters
📊 Insights
Theft and battery are most common crimes
Crime is concentrated in certain districts
Arrest rate is lower compared to total crimes
Crime trends vary over time
🛠 Tools Used
Power BI
Data Cleaning
Data Visualization
📂 Dataset

Chicago Crime Dataset (Open Data)
